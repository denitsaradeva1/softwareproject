# Smart Vehicle Management and Analytics Simulator

## Description
Smart Vehicle Management and Analytics Simulator is a Python-based web application for vehicle monitoring, maintenance tracking, risk analysis, and cost forecasting.

The system allows users to:
- add and manage vehicles,
- analyze vehicle health and risk,
- generate maintenance recommendations,
- track operational costs,
- visualize analytics dashboards.

The project also integrates a Kaggle vehicle dataset for fleet analytics and predictive maintenance simulation.
